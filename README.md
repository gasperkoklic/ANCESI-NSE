
# ANCESI (Altium to Neoden Converter Editor with Stock Integration) 
ANCESI is a python script that offers a GUI environment for converting PnP CSV files made using Altium Designer into PnP CSV files formatted in a manner that the NeoDen YY1 PnP machine will understand. It also offers the functionality of editing already converted PnP CSV files.

# NSE (NeoDen Stock Editor) 
NSE is a python script that offers a GUI environment for creating CSV files that contain stock information for your deployment of the NeoDen YY1. These stock files have a direct integration with ANCESI and speed up work with ANCESI greatly by eliminating the need to constantly consult what feeder number holds what component. NSE also offers the ability to edit already existing stock CSV files made by NSE.
